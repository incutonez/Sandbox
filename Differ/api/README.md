# Description

This project uses NestJS to host some dummy endpoints for the UI to interact with.  It also uses shared models. 

## Installation

```bash
$ npm i
```

## Running the app

```bash
# watch mode
$ npm run start:dev
# navigate to localhost:3000/diff

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
